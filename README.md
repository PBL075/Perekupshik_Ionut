# Perekupshik_Ionut
Aplicație simplă în C# pentru calcularea profitului din revânzarea telefoanelor.

Ce face programul:
Cere prețul de achiziție al telefonului.
Cere prețul la care a fost vândut.
Afișează profitul net realizat.

Cum se rulează:
Descarcă fișierul Program.cs.
Rulează-l folosind dotnet run în terminal.
